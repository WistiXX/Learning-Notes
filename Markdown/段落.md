# 字体
## 1. 斜体
*斜体文本*

文本前后加**一个** \* 或 \_
\*斜体文本\*
\_斜体文本\_
## 2. 粗体
**粗体文本**

文本前后加**两个** \* 或 \_
\*\*粗体文本**
\_\_粗体文本\__
## 3. 粗斜体
***粗斜体文本***

文本前后加**三个** \* 或 \_
\*\*\*粗斜体文本***
\_\_\_粗斜体文本___

# 脚注
Makedown脚注格式如下：
**\[^文本内容]**

[^内容出处]创建脚注格式类似这样。

**↓要加这条分隔线**

[^出处]: 这是脚注指向的文本
