# 1. 算术运算符
## 1. 相加（ + )
```go
var V int
A, B = 5, 2
V = A + B
fmt.Print(V)
```
**输出为：`7`**
***
## 2. 相减（ - ）
```go
var V int
A, B = 5, 2
V = A - B
fmt.Print(V)
```
**输出为：`3`**
***
## 3. 相乘（ * ）
```go
var V int
A, B = 5, 2
V = A * B
fmt.Print(V)
```
**输出为：`10`**
***
## 4. 相除（ / ）
`/`会直接截断小数部分，所以不会四舍五入，值1.666...结果会是1
```go
var V int
A, B = 5, 2
V = A / B
fmt.Print(V)
```
**输出为：`2`**
***
## 5. 求余（ % ）
规则与`/`
```go
var V int
A, B = 5, 2
V = A % B
fmt.Print(V)
```
**输出为：`1`**
***
## 6. 自增（ ++ ）
```go
var V int = 1
V++
fmt.Print(V)
```
**输出为：`2`**
***
## 7. 自减（ -- ）
```go
var V int = 1
V--
fmt.Print(V)
```
**输出为：`0`**
